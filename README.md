## Daftar Script
- `identitas.sh` - Input nama & umur, kasih sapaan sesuai umur
- `hitung.sh` - Kalkulator sederhana (tambah, kurang, kali, bagi)
- `cek-file.sh` - Cek file, tampilkan info dan 3 baris pertama
- `auto-backup.sh` - Backup otomatis folder
- `monitoring-client.sh` - Client monitoring
- `monitoring-server.sh` - Server monitoring
